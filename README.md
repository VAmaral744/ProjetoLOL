# Projeto LoL

## Sobre o projeto
Este projeto foi desenvolvido com o objetivo de **aprender e aplicar Maven e trabalhar com APIs** em Java.  
Ele utiliza a **API Data Dragon da Riot Games** (HTTPS) para consultar dados de campeões e itens do jogo **League of Legends**.

O objetivo principal é analisar os dados coletados para **identificar matematicamente o melhor campeão e item para cada posição**.

---

## Tecnologias utilizadas
- Java 21
- Maven
- API Data Dragon (Riot Games)
- Jackson (para manipulação de JSON)

---

## Status do projeto
Atualmente estou na fase de **implementação com Maven**.  
Tenho dedicado cerca de duas semanas ao aprendizado de Maven e, apesar de alguns desafios iniciais, estou conseguindo **progredir gradualmente**.
